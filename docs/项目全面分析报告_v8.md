## 一、项目概述

### 1.1 项目定位

ITops Agent 是一个**全栈式 AIOps 运维平台**，集成：

- **传统运维**：服务器管理、SSH 终端、容器/VM、Kubernetes、网络设备、数据中心
- **AIOps 能力**：LLM Agent、根因分析（RCA）、AI 修复建议、知识库、多 Agent 协同
- **告警与监控**：多源告警接入、降噪、关联分析、自动响应、大屏可视化
- **MCP 工具协议**：42 个内置工具，支持外部 MCP Server 接入
- **自动化运维**：修复策略、自动伸缩、工作流编排、定时任务

### 2.2 入口与组装

#### app.ts 初始化流程（`initApp()` async 函数）

```

2. http.createServer(app) + SocketIOServer(path=/socket.io/)
3. setIOInstance(io) + setupWebSocket(io) + vncProxyService.initialize(io)
4. container.register('io', () => io, [])   # io 实例注册到 DI
5. initAllServices()                 # 服务初始化（拓扑排序）

7. setupSwagger(app)                 # Swagger 文档
8. httpServer.listen(PORT)           # 默认 3001
```
